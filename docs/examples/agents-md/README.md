# AGENTS.md Anthology

This anthology contains three anonymized AGENTS.md examples to build pattern recognition across team scales.

## Entries
- Startup: [`startup.md`](./startup.md)
- YC-scale: [`yc-scale.md`](./yc-scale.md)
- Enterprise guardrailed: [`enterprise.md`](./enterprise.md)

## Compare-and-Contrast Lens
- Risk posture: how much failure cost is tolerated before escalation.
- Autonomy level: what actions are delegated vs gated.
- Kill-switch strategy: trigger conditions, owners, and rollback clarity.
- Merge-cap discipline: whether velocity controls are static or evidence-based.

## Weak Example Prompt (Pattern Spot Exercise)
Use this deliberately weak sample and identify at least five critical gaps:

```md
# AGENTS
Use AI to move fast.
Do your best.
Ship quickly.
```

Reference answer themes:
- No tooling boundaries
- No validation requirements
- No escalation logic
- No incident or kill-switch controls
- No ownership or release gating
